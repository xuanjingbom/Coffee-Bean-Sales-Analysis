# Coffee Bean Sales Analysis

## Overview

This project analyzes coffee bean sales data to identify sales patterns, customer purchasing behavior, and business performance trends.

The objective is to generate meaningful insights from sales data through data cleaning, visualization, and exploratory analysis.

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Data Visualization

## Files

* Final Report.pdf
* Presentation Slides.pdf
* Source Code.ipynb
* Dataset.csv

## Key Skills Demonstrated

* Data Cleaning
* Exploratory Data Analysis (EDA)
* Business Insight Generation
* Data Visualization
* Report Writing
